Customers Customer Rest Controller
/api/customers
GET
/all Returns all customer information available
GET
/by-id Returns customer information by ID Requirement(s): [param: id (Integer)]
GET
/by-name Returns customer information by name Requirement(s): [param: name (String)]
GET
/by-phone Returns customer information by phone Requirement(s): [param: phone (String)]
GET
/by-email Returns customer information by email Requirement(s): [param: email (String)]
POST
/new Adds new customer to the database and returns Requirement(s): [body: Customer]
PATCH
/update Updates customer information by id and returns Requirement(s): [param: id, body: Customer]

Orders Order Rest Controller
/api/orders
GET
/all Returns all order information available
GET
/by-order-id Returns order information by order ID Requirement(s): [param: order-id (Integer)]
GET
/by-customer-id Returns order information by customer ID Requirement(s): [param: customer-id (Integer)]
GET
/by-product-id Returns order information by product ID Requirement(s): [param: product-id (Integer)]
GET
/by-paid Returns order information by paid status Requirement(s): [param: paid (Boolean)]
GET
/by-placed-on Returns order information on date Requirement(s): [param: date (Date)]
GET
/by-placed-month Returns order information in month Requirement(s): [param: month (Integer)]
GET
/by-placed-year Returns order information in year Requirement(s): [param: year (Integer)]
GET
/by-price-range Returns order information within price range Requirement(s): [param: min, max (Double)]
POST
/new Adds new order information to the database and returns Requirement(s): [body: Order]
PATCH
/update Updates order information by id and returns Requirement(s): [param: id, body: Order]
DELETE
/delete Deletes order information by id Requirement(s): [param: id]
